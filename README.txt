一期开发的主要工作是设计客服机器人的基础架构，做到功能模块可扩展、处理方法可优化，一期的主要工作如下几个方面：
1、 开发环境的搭建
    项目采用Python语言，主要的开发环境为Linux+web.py，首先根据微信协议，搭建开发的基础环境
2、 客服知识库的建立
    客服知识整理为Excel，需要实现开发语言读取Excel文件，并根据具体的格式导入到Redis的hset中。
3、 用户输入内容分词和匹配度计算
    项目的核心，采用jieba分词器对用户提问的问题进行分词，一期只支持文本形式的提问。设定匹配思路，实现匹配功能。
4、 客服机器人转人工客服的功能设计
    当用户输入的问题客服机器人不能匹配时，用户返回聊天对话框的网址，触发人工客服。聊天对话网站的设计由赵超越负责。
5、 聊天记录的保存
    聊天记录主要保存在MySQL中。
6、 客服机器人的检测
    为了维护机器的高稳定性，设定Shell脚本检测客服机器人服务是否存活，并设计自动重启脚本。
